Welcome to Fuze Page

FuzeVD

A Utility to download youtube videos for free into video or audio.

Go to the FuzeVD page for more download options.

Fuze Game

This web game runs on Fuze Engine, a game engine which uses unicode text to render objects.
At the moment there are 4 levels, with objectives to eliminate different enemies for each one.
The player can equip two weapons out of a selection at the start of each level. The same weapon can be picked for double fire rate.
